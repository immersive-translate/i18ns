
# Stripe 계정 잔액 소개

## 계정 잔액 (Stripe 고객 청구서 잔액) 이란 무엇인가요?

구독을 조정하거나, 일부 서비스를 취소하거나, 초과 금액이 발생했을 때 (환불 작업이 아닌 경우), 초과 부분은 즉시 Stripe 계정 잔액 (Customer Invoice Balance) 에 입금됩니다.
이는 Stripe 플랫폼에서의 잔액과 동일하며, 향후 소비에 상계할 수 있습니다.

## 왜 잔액이 발생하나요?

구독을 다운그레이드하고, 새로운 플랜 요금이 더 낮은 경우 → 차액이 계정 잔액으로 환불됩니다.

플랜을 조기에 취소한 경우 → 비례 계산된 미사용 요금이 자동으로 계정 잔액으로 환불됩니다.

이전에 과다 지불한 경우 → 초과 지불 금액이 이 잔액에 입금됩니다.

## 계정 잔액은 어떻게 사용되나요?

다음 청구 주기 (예: 월간 갱신) 에서 시스템은 계정 잔액을 우선적으로 사용하여 공제합니다.

잔액이 충분한 경우, 추가 결제가 필요하지 않습니다.

잔액이 부족한 경우, 차액만 지불하면 됩니다.

## 계정 잔액이 만료되나요?

아니요. 잔액은 완전히 사용될 때까지 계정에 유지됩니다.

## 계정 잔액을 확인하는 방법은?

개인 센터 → 구독 관리 → 다음 결제의 상세 보기에서 애플리케이션의 계정 잔액을 확인할 수 있습니다.
