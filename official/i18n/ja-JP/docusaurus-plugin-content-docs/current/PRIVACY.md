# Immersive Translate プライバシーポリシー

最終更新日：2026-04-16

## はじめに

本 Immersive Translate プライバシーポリシー（以下「本ポリシー」という）は、Funstory.ai Limited（香港）およびその関連会社（以下「当社」「私たち」「我々」または「当社」といいます。）が提供する Immersive Translate サービス（以下「本サービス」といいます。）に適用されます。本サービスには第三者のウェブサイトへのリンクが含まれる場合があります。Immersive Translate は、かかる第三者ウェブサイトのプライバシー慣行または内容について責任を負いません。当該ウェブサイトのプライバシーに関する記述をよくご確認になることをお勧めします。

本ポリシーは、統一的な世界的な個人情報保護の枠組みを示すことを目的とし、管轄区域ごとの適用される個人情報保護関連法令に従って調整および解釈されます。本サービスを利用する前に、当社がお客様の個人情報をどのように取り扱うかについて本ポリシーをよくお読みください。当社は、Immersive Translate の製品およびサービスを利用される方（総称して「ユーザー」または「お客様」といいます。）の個人情報およびプライバシーを守ることに尽力します。本ポリシーを通じて、当社がお客様の個人情報を収集し利用する際のルールを説明し、お客様の権利および利益をよりよく保護します。

## データ管理者

## 1. データの収集および利用

### お客様から当社に提供される情報

- 当社が収集するのは電子メールアドレスに限られ、主にアカウントの識別、サインイン、パスワードの再設定、およびフォローアップの製品アップデート体験へのご招待のために利用します。
- Google、Apple、WeChatなどの第三者アカウントでサインインされる場合、当社はよりよいサインイン体験を提供するため、それら第三者が提供する OpenID および電子メールアドレスを収集します。
- Immersive Translate のファイル／画像翻訳機能をご利用になる場合、翻訳中に入力いただいたコンテンツ（画像およびファイルを含みます）を収集します。当該情報はお客様の個人情報とは紐づけられず、より便利で高品質な翻訳結果をご提供するため、翻訳サービスの改善および提供のみに利用します。
- Immersive Translate の BabelDoc 翻訳機能をご利用になる場合、翻訳品質向上を主な目的として、お客様がアップロードしたファイルの内容を収集し保存することがあります。お客様から提出されたテキストまたはファイルは恒久的には保存せず、翻訳または改善の提供・送信に必要な範囲でのみ一時的に保持し、個人情報とは紐づけません。契約サービスが完全に履行された後、提出されたすべてのテキストまたはファイルならびにそれらの翻訳または改善は削除されます。
- 支払いを処理するにあたり、当社は第三者の決済サービスプロバイダーを利用してお客様の支払い情報を処理する場合があります。当社は銀行またはクレジットカードの完全な情報を直接保管しません。かかる情報は業界のセキュリティ基準を満たす決済サービスプロバイダーにより処理されます。

### 当社が自動的に収集する情報

- アプリケーションの利用状況（利用頻度やご利用機能など）に関する情報を収集します。かかるデータは集約されており、特定の個人を直接特定することを目的としたものではありません。
- ウェブページ翻訳の利用において、翻訳の内容はお客様が選択した翻訳エンジンにのみ送信されます。より便利で高品質な体験を提供しサービスを改善するため、当社は利用統計（翻訳頻度、機能の利用、その他疑似名化情報など）のみを収集します。
- Immersive Translate をご利用になる際に、翻訳中に発生したエラーログを収集および保存し、障害調査およびサービス体験の改善に活用します。
- サービス改善のため、デバイス情報（デバイス ID、ブラウザの種類およびバージョンなど）を収集します。デバイス ID は疑似匿名識別子として利用され、お客様を直接特定することを目的としたものではありません。
- お客様の IP アドレスを収集し、国または地域を判定する場合があります。かかる情報は、現地語のコンテンツの提供、製品の最適化、サービスセキュリティの確保、および不正利用の防止のみに利用します。

当社は利用状況の把握およびサービスの改善のため、Google アナリティクスおよび自社運用のアナリティクスサービス（例：analytics.immersivetranslate.com）を利用して情報を収集します。かかる情報は通常、集約および／または疑似名化（実現可能な範囲で非識別化）されます。ブラウザの設定または関連ツールを通じて、必須でない解析用クッキーを管理またはオプトアウトすることができます。当社はお客様を直接特定するためにそれを利用しません。

## データの保存期間

当社は、本ポリシーに記載された目的を達成するために必要な期間に限り個人情報を保持します。例：

- アカウント情報は、一般にアカウントがアクティブである限り保持されます。
- ログおよびセキュリティ記録はシステムのセキュリティ確保に用いられ、合理的な期間経過後に削除または疑似名化されます。

## 2. データの共有

当社は、お客様に本サービスを適切に提供するために必要な範囲で、アップロードされた翻訳ファイル、注文データ、およびデバイスデータを関連会社および第三者（以下のパートナー種別）と共有することがあります。当社のパートナーには次のようなものがあります。

- 購読者に PDF 文書翻訳サービスを提供する際にお客様がアップロードした PDF ファイルは暗号化のうえ送信され、文書コンテンツの抽出および翻訳に必要な機械判読テキストの生成のため、処理を第三者サービスプロバイダーである Mathpix に委託します。この過程において当社はデータ最小化の原則に従い、必要最小限のファイルのみを送信します。Mathpix には契約上の義務に基づき、業界標準のセキュリティ対策により文書を保護するよう求めます。処理後、Mathpix は抽出されたコンテンツを当社に返送し、当社が翻訳結果を生成します。Mathpix の詳細はその [プライバシーポリシー](https://mathpix.com/privacy) をご覧ください。なお、法律で求められる場合を除き、当社はお客様の個人情報または個人データを第三者のベンダーまたはパートナーと共有しません。

### 2.2 第三者翻訳サービス

- 当社は第三者の翻訳サービスを利用します。翻訳の際には、翻訳に関連するコンテンツを当該サービスに送信する必要があります。現在の当社の知識および関連プロバイダーとの契約ならびにその公表明細に基づき、当該サービスプロバイダーはかかるコンテンツを自社モデルのトレーニングに利用していないものと認識しています。

当社は翻訳機能を提供するために必要な範囲でのみコンテンツを共有し、個人を識別可能な情報を収集することが目的となるよう翻訳サービスを設計してはおりません。当社はデータのセキュリティを守るために適切な技術的および組織的対策を実施します。

### 2.3 金融機関および第三者決済提供者

- 当初の目的のために情報がもはや必要でない；

## 6. クッキーに関する条項

### クッキーとは

「クッキー」とは、ウェブサーバーが生成しユーザーのデバイスに保存される小さなデータです。

### 当社が使用するクッキー

当社はセッションクッキーおよび永続的クッキーを使用します。セッションクッキーはブラウザを閉じると削除されますが、永続的クッキーはデバイスに残り、次に当社のサイトを訪れた際に活性化され、手動でも削除できる場合があります。一般に永続的クッキーはデバイス、プログラム、またはファイルを損ないません。

### クッキーの管理方法

ブラウザの設定により、すべてまたは一部のクッキーを拒否できます。手順についてはご利用ブラウザの説明を参照してください。ただし、すべてのクッキー（厳密に必要なクッキーを含む）をブロックすると、当社のサイトまたは特定のコンテンツにアクセスできないことがあります。

## 7. Safari 拡張機能の権限に関する説明

Safari で Immersive Translate 拡張機能を有効にすると、次のような権限の通知が表示されることがあります。

「この拡張機能は、銀行カード詳細などの機密情報の読み取りと変更ができます。」

これが懸念を招き得ることは理解しており、以下に背景およびプライバシーにおける約束を示します。

### 7.1 この通知の意味は何か

この通知は、閲覧中に翻訳サービスを提供するために、翻訳拡張機能がウェブページの内容を読み取り変更する権限が必要であることを意味します。拡張機能はご覧になっているテキストを処理し、お客様が選択した言語に翻訳します。

「機密情報」について：技術的には拡張機能はウェブページ上のすべてのコンテンツ（機密データを含み得ます）を読み取れる能力を有しますが、当社の翻訳拡張機能はテキストの翻訳に重点を置き、銀行カード詳細などのプライベート情報を意図的に収集または利用しません。

### 7.2 なぜ Safari にのみこの通知が表示され、Chrome、Edge、その他には表示されないのか

主にブラウザ間でのプライバシーポリシーおよび通知の仕組みの違いによるものです。Safari は、実運用での利用がごく限定的であっても、可能な権限すべてについてユーザーに事前に知らせる傾向があります。Chrome は、開発者が権限の使い道を開示している場合、同様に目立つ通知を出さないことがあります。Safari の方針は、極めてまれな事例でのみまたはまったく使わない権限であっても、インストール済み拡張機能が請求する権限についてユーザーが十分認識できるようにすることが意図されています。

### 7.3 これらの権限をどのように利用しているか

ウェブページの内容へのアクセスおよび変更権限は、ご覧になっているウェブページを対象言語に翻訳するためのみに用いられます。Safari または Chrome にかかわらず、プライバシーを重視し、やむを得ない場合以外に追加の権限は請求しません。どのブラウザをご利用でも、データの収集、利用および保護方法をよりよく理解するために本ポリシーをご一読いただくことをお勧めします。本ポリシーに関するご質問は [support@immersivetranslate.com](mailto:support@immersivetranslate.com) までご連絡ください。

本付録は、大韓民国の個人情報保護法（「PIPA」）が、大韓民国内で提供されるサービス、または大韓民国に所在するユーザーに対して適用される場合に適用されます。

### D.1 処理の目的および適法性

当社は、特定され明示的かつ適法な目的に限り個人情報を処理し、PIPA の下で必要な同意またはその他の適法な根拠を取得します。法律で別段の定めがある場合を除き、収集時に開示された目的を超えて個人情報を利用しません。

### D.2 委託および第三者への開示

当社が第三者に個人情報の処理を委託する場合（クラウドサービス、翻訳、決済、技術支援などを含む。）、適用法に従い次を行います。

- 受託者と書面契約を締結する；
- 目的、範囲、および技術的・組織的セキュリティ対策を明記する；
- 受託者の個人情報処理について継続的に監督する。

法令により許容される第三者への個人情報の提供においては、事前にご通知し、法により求められる場合には別途同意を取得します。

### D.3 越境移転

個人情報が大韓民国以外へ移転される場合、PIPA に従い次を行います。

- 越境移転の目的、受領者、項目、保存期間を事前にご通知；
- 法律により求められる場合には別途同意を取得；
- 海外の受領者に対し PIPA に適合したセキュリティ対策の継続的実施を求める。

### D.4 ユーザー権利

PIPA が適用される場合、ユーザーは法的に次を請求できます。

- 自身の個人情報へのアクセス；
- 不正確または不要な個人情報の訂正または削除；
- 個人情報処理の停止；
- 既に付与した同意の撤回。

当社は法令に従い、合理的な期間内にこれらの請求に対応します。

### D.5 セキュリティ対策

当社は PIPA および関連規則に基づき、次を含む適切な技術的・管理的・物理的セキュリティ対策を実施します。

- アクセス制御および権限管理；
- 暗号化または非識別化；
- 内部管理体制および職員研修；
- セキュリティインシデント対応および記録の保持。

### D.6 個人情報侵害の通知

個人情報のセキュリティに影響し得る漏えい、改ざん、または紛失が発生した場合、当社は PIPA に従い速やかに是正措置を講じ、法令により求められる場合には次を実施します。

- 韓国の個人情報保護委員会（PIPC）への報告；
