# 二言語ページを共有する方法は？

ユーザーの皆様へ

「二言語ページ共有」機能は2025年9月15日をもって正式に終了し、ご利用いただけなくなりました。
これまでのご支援とご理解に心より感謝申し上げます。今後もより良いサービスをご提供できるよう、製品の改善に努めてまいります。

ご不明な点がございましたら、[support@immersivetranslate.com](mailto:support@immersivetranslate.com) までお問い合わせください。
