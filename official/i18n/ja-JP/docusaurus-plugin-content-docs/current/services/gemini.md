# ニウトランス翻訳

[公式 Niu 翻訳ドキュメント](https://niutrans.com/documents/contents/question/1)を直接ご確認ください。

## Gemini API キーの取得

- https://makersuite.google.com/ を開く
- Google アカウントでサインイン
- `Get API Key` -> `Create API key in new project`をクリックしてキーを保存
- その後、Immersive Translate 拡張機能の Gemini 設定セクションにキーを入力します。

## 注意事項

- 現在、このサービスは香港の IP をサポートしていません。[サポートされている地域を表示](https://ai.google.dev/available_regions)
- 一部のユーザーからのフィードバックによると、Immersive Translate で Gemini を翻訳に使用する際、IP(共有 IP など) または頻度の理由により、Google Cloud によって悪用と判断され、API キーまたは Google Cloud アカウントが無効になる可能性があります。慎重に使用するか、サブアカウントの使用を検討してください。
