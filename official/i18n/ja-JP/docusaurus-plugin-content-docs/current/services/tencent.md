# テンセント翻訳君

## 簡単な説明

1. 公式サイト:[機械翻訳 _ インテリジェント翻訳 _ 自動翻訳 - Tencent Cloud](https://cloud.tencent.com/product/tmt)
2. 公式料金説明:[機械翻訳課金概要-購入ガイド-ドキュメントセンター-Tencent Cloud](https://cloud.tencent.com/document/product/551/35017)
3. 有料版は毎月最初の500万文字まで無料で、それを超えると100万文字あたり58ドルが課金されますので、誤って課金されないように使用量の制限にご注意ください。

## 申請手順

1. [Tencent Cloudウェブサイト](https://cloud.tencent.com/)を開いてログインします。ログイン成功後、ページ右上のアバターにマウスを移動し、「アカウント情報」を選択して個人認証を行います。Tencent Cloud機械翻訳を使用するには個人認証が必要ですが、既に認証済みの場合はスキップできます。
2. [機械翻訳_インテリジェント翻訳_自動翻訳-Tencent Cloud](https://cloud.tencent.com/product/tmt)を開き、「今すぐ使用」ボタンをクリックします。ログイン後、Tencent機械翻訳サービスコンソールに移動します。
3. 有料版を開くことを選択します。
4. アクセスキーを作成します。ウェブページ右上のアバターにマウスをホバーし、[アクセス管理](https://console.cloud.tencent.com/cam/overview)を選択してから、左メニューで[アクセスキー - API キー管理](https://console.cloud.tencent.com/cam/capi)を選択します。キーを直接作成しない方が良いです。なぜなら、メインアカウントで作成したキーはアカウント内のすべてのリソースにアクセスできるため、安全のためにサブアカウントを作成することを選択してください。「ユーザー権限」項目で「機械翻訳」を検索し、これだけをチェックします。
5. 作成が成功すると、このサブアカウントの「SecretId」と「SecretKey」が表示されます。これを拡張機能に入力してください！
6. 完了です 🎉 疑問がある場合は、[こちら](https://github.com/immersive-translate/immersive-translate/issues/137)でフィードバックをお願いします。
