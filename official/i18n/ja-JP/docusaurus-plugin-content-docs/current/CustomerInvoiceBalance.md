
# Stripe アカウント残高の紹介

## アカウント残高（Stripe 顧客請求書残高）とは？

サブスクリプションを調整したり、一部のサービスをキャンセルしたり、余剰金額が発生した場合（返金操作以外）、余剰分は即座に Stripe アカウント残高（Customer Invoice Balance）に預け入れられます。
これは Stripe プラットフォーム上の残高に相当し、今後の消費に充当できます。

## なぜ残高が発生するのか？

サブスクリプションをダウングレードし、新しいプランの料金が低い場合 → 差額がアカウント残高に返金されます。

プランを早期にキャンセルした場合 → 比例計算された未使用料金が自動的にアカウント残高に返金されます。

以前に過払いした場合 → 過払い金額がこの残高に預け入れられます。

## アカウント残高はどのように使用されますか？

次の請求サイクル（例：月次更新）では、システムがアカウント残高を優先的に使用して差し引きます。

残高が十分な場合、追加の支払いは不要です。

残高が不足している場合、差額のみを支払う必要があります。

## アカウント残高は期限切れになりますか？

いいえ。残高は完全に使用されるまでアカウントに保持されます。

## アカウント残高を確認する方法は？

個人センター → サブスクリプション管理 → 次回支払いの詳細表示で、アプリケーションのアカウント残高を確認できます。
