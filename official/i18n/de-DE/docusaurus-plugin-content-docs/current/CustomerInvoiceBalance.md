
# Einführung in Stripe-Kontostand

## Was ist Kontostand (Stripe-Kundenrechnungsguthaben)?

Wenn Sie Ihr Abonnement anpassen, bestimmte Dienste kündigen oder überschüssige Beträge haben (keine Rückerstattungsvorgänge), wird der überschüssige Betrag sofort auf Ihr Stripe-Kontoguthaben (Customer Invoice Balance) eingezahlt.
Es entspricht Ihrem Guthaben auf der Stripe-Plattform und kann zur Verrechnung zukünftiger Ausgaben verwendet werden.

## Warum entsteht ein Guthaben?

Sie stufen Ihr Abonnement herab und der neue Plan hat niedrigere Gebühren → Die Differenz wird auf Ihr Kontoguthaben zurückerstattet.

Sie kündigen einen Plan vorzeitig → Die anteilig berechneten ungenutzten Gebühren werden automatisch auf Ihr Kontoguthaben zurückerstattet.

Sie haben zuvor zu viel gezahlt → Der überzahlte Betrag wird auf dieses Guthaben eingezahlt.

## Wie wird das Kontoguthaben verwendet?

Während des nächsten Abrechnungszyklus (z. B. monatliche Verlängerung) verwendet das System vorrangig das Kontoguthaben für die Verrechnung.

Wenn das Guthaben ausreicht, ist keine zusätzliche Zahlung erforderlich.

Wenn das Guthaben nicht ausreicht, müssen Sie nur die Differenz zahlen.

## Läuft das Kontoguthaben ab?

Nein. Das Guthaben bleibt auf Ihrem Konto, bis es vollständig verwendet wurde.

## Wie kann ich das Kontoguthaben einsehen?

Im Persönlichen Zentrum → Abonnement verwalten → Details der nächsten Zahlung anzeigen können Sie das Kontoguthaben der Anwendung einsehen.
