
# Stripe 帳戶餘額介紹

## 帳戶餘額（Stripe 客戶帳單餘額）是什麼？

當您調整訂閱、取消某些服務，或產生多餘的金額時（非退款操作），多出的部分會立即存入您的 Stripe 帳戶餘額（Customer Invoice Balance）。
它相當於您在 Stripe 平台上的餘額，可用於未來消費抵扣。

## 為什麼會產生餘額？

您降級訂閱，新的計劃費用更低 → 差額會退回到帳戶餘額。

您提前取消某計劃 → 按比例計算的未使用費用自動退回到帳戶餘額。

您之前支付過多 → 超付金額會存入此餘額中。

## 帳戶餘額會如何使用？

在下一次帳單結算時（例如月度續費），系統會優先使用帳戶餘額進行抵扣。

若餘額足夠，則無需額外付費。

若餘額不足，則只需支付差額。

## 帳戶餘額會過期嗎？

不會。餘額會一直保留在您的帳戶中，直到完全使用。

## 如何查看帳戶餘額？

個人中心->管理訂閱->下筆付款的查看詳情中，應用的帳戶餘額可以查看。
