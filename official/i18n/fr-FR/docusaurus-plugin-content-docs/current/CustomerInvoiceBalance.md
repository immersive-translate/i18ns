
# Introduction au Solde de Compte Stripe

## Qu'est-ce que le Solde de Compte (Solde de Facture Client Stripe) ?

Lorsque vous ajustez votre abonnement, annulez certains services ou avez des montants excédentaires (opérations non remboursables), la partie excédentaire sera immédiatement déposée sur votre solde de compte Stripe (Customer Invoice Balance).
Il équivaut à votre solde sur la plateforme Stripe et peut être utilisé pour compenser la consommation future.

## Pourquoi un solde se produit-il ?

Vous rétrogradez votre abonnement et le nouveau plan a des frais plus bas → La différence sera remboursée sur votre solde de compte.

Vous annulez un plan à l'avance → Les frais non utilisés calculés proportionnellement seront automatiquement remboursés sur votre solde de compte.

Vous avez précédemment surpayé → Le montant surpayé sera déposé sur ce solde.

## Comment le solde de compte est-il utilisé ?

Lors du prochain cycle de facturation (par exemple, renouvellement mensuel), le système utilisera en priorité le solde de compte pour la déduction.

Si le solde est suffisant, aucun paiement supplémentaire n'est requis.

Si le solde est insuffisant, vous n'avez qu'à payer la différence.

## Le solde de compte expire-t-il ?

Non. Le solde restera sur votre compte jusqu'à ce qu'il soit entièrement utilisé.

## Comment consulter le solde de compte ?

Dans Centre Personnel → Gérer l'Abonnement → Voir les Détails du Prochain Paiement, vous pouvez consulter le solde de compte de l'application.
