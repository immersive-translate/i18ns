# 浏览器插件集成

## 简介

Immersive Translate 现已支持 **多种主流浏览器**，包括 Chrome、Edge、Firefox、Safari 等。无论用户使用哪种浏览器，都能快速安装并享受沉浸式双语翻译体验。

我们的插件能够帮助用户更高效地阅读外文内容，提升学习与工作效率，是跨语言阅读的最佳工具。

## 集成优势

* **多浏览器兼容**：一次集成，覆盖全平台，减少适配工作量。
* **功能用户友好**：界面简洁、交互直观，用户零门槛上手。
* **网页场景兼容性强**：支持多种网站和应用场景，包括新闻、社交媒体、科研论文、在线课堂等，确保沉浸式体验。
* **持续迭代更新**：插件功能和翻译引擎保持最新，保障长期稳定可用。
