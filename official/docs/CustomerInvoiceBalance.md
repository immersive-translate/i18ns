
# Stripe 账户余额介绍

## 账户余额（Stripe 客户账单余额）是什么？

当您调整订阅、取消某些服务，或产生多余的金额时（非退款操作），多出的部分会立即存入您的 Stripe 账户余额（Customer Invoice Balance）。
它相当于您在 Stripe 平台上的余额，可用于未来消费抵扣。

## 为什么会产生余额？

您降级订阅，新的计划费用更低 → 差额会退回到账户余额

您提前取消某计划 → 按比例计算的未使用费用自动退回到账户余额

您之前支付过多 → 超付金额会存入此余额中

## 账户余额会如何使用？

在下一次账单结算时（例如月度续费），系统会 优先使用账户余额进行抵扣

若余额足够，则无需额外付费

若余额不足，则只需支付差额

## 账户余额会过期吗？

不会。余额会一直保留在您的账户中，直到完全使用。

## 如何查看账户余额？

个人中心->管理订阅->下笔付款的查看详情中，应用的账户余额可以查看。
